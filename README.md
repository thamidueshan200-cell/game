# How to Play

## player 01

    -A = Move Left
    -D = Move Right
    -W = Jump
    -SPACE = Attack

## player 02

    -Right Arrow = Move Right
    -Left Arrow = Move Left
    -Arrow UP = Jump
    -Arrow Down = Attack
